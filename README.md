# Deposito

A Vite + React + TypeScript + Tailwind CSS project.

## Development

Prerequisites:
- Node.js 18+
- npm or bun

Install dependencies and start the dev server:

```sh
npm i
npm run dev
```

## Tech Stack
- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Build
```sh
npm run build
```

## Preview build
```sh
npm run preview
```
